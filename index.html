A little bit about me

I'm a husband, father, Spanish>English translator, project manager and back-end developer.

I love all things tech...

and after spending many hours watching developer conferences and tech keynotes online, I had a thought...why not spend my days doing what I already spend my personal time doing? So began my coding journey.

After picking up HTML and CSS online, I searched for a place in the development world where my organization skills and appreciation for spreadsheets could be put to good use. Enter Java; not only one of the most ubiquitous coding languages, but also one of the best languages to use to build strong, scalable databases. And thus, my love affair for object-oriented programming (and the compiler) began.

Some of my apps

I have thoroughly enjoyed learning Java. It's such a powerful language that can be applied in so many different ways.

Community

An invitation-only social networking app with a Java back-end and iOS front-end. This app allows the user to be invited into organizations and interact with the events and posts associated with the organizations of which they are a member.

The Java back-end is a set of web services that connect the relational Postgres database to the iOS front-end.

I also designed the front-end test site using AngularJS, which was created to test end-to-end connection of features and review JSON data transfer prior to connecting to the iOS end.

Community Repo

Bank

A complex banking system that includes polymorphism to define different account types, threading to accrue interest, and I/O for file saving.

The system supports any number of users, and users can create any number of accounts. .

Web Chat Client

A web-based chat client built with Hibernate, a Spring controller, a Spring Rest controller, HTML/CSS and AngularJS.

This is my first time blending all of these tools together to make a web-based chat system.

Given my...distaste...for front-end, I'd say it's pretty nice!

Chat App Repo

ASCII Art

This is the first Java app I ever designed. It's an ASCII art project that pays homage to my beloved Atlanta Braves.

During my training at The Iron Yard, things got very intense and I wasn't sure that I was going to make it. Sometimes, when things got bad, I would go back to old assignments and marvel at just how far I'd come.

During an especially tough time, I opened this project and realized I manually chopped the tomahawk through the Braves logo three times. Realizing I now had more tools at my disposal, I deleted the extra chops and added a for loop to automate the number of chops.

That exercise helped me realize just how much I had grown as a programmer in a short period of time and gave me the strength to carry on.

It will always hold a special place in my heart.
